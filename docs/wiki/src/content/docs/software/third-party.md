AntBot 구동을 위해 활용된 외부 오픈소스 및 하드웨어 제조사 제공 패키지입니다.

| **패키지**  | **출처** | **설명**                                        |
| ---------------- | ----------------- | ----------------------------------------------- |
| [DynamixelSDK](https://github.com/ROBOTIS-GIT/DynamixelSDK/tree/humble)     | ROBOTIS           | Dynamixel Protocol 2.0 통신용 표준 SDK          |
| [coin_d4_driver](https://github.com/ROBOTIS-GIT/coin_d4_driver/tree/humble)   | ROBOTIS           | COIN-D4 2D LiDAR 데이터 수신 및 처리 드라이버   |
| [vanjee_lidar_sdk](https://github.com/ROBOTIS-move/antbot/tree/main/vanjee_lidar_sdk) | Vanjee     | Vanjee 3D LiDAR ROS 2 표준 드라이버             |
| [vanjee_lidar_msg](https://github.com/ROBOTIS-move/antbot/tree/main/vanjee_lidar_msg) | Vanjee     | Vanjee LiDAR 데이터용 커스텀 메시지 정의 패키지 |
| [ublox](https://github.com/KumarRobotics/ublox/tree/ros2)            | KumarRobotics     | u-blox GPS/GNSS 센서 ROS 2 드라이버             |
| [OrbbecSDK_ROS2](https://github.com/orbbec/OrbbecSDK_ROS2/tree/v2-main)   | Orbbec            | Orbbec RGB-D 카메라 ROS 2 드라이버              |
