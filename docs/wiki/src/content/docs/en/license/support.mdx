## Support Scope

ROBOTIS AI provides technical support for AntBot's **hardware and officially provided software**.

| Support Item | Description |
| :--- | :--- |
| Hardware Defects and Malfunctions | Physical defects and malfunctions of the robot body, frame, and electrical components |
| Drive System | Dynamixel motor hardware issues |
| Battery and Charging System | Battery status, charger, and charging dock issues |
| RCU Board and HMI Panel | Control board and operation panel issues |
| Official Software and Firmware | Issues related to officially provided software and firmware updates |

:::caution
Technical support is **not available for unauthorized hardware modifications**. This includes frame modifications, unofficial parts replacement, and electrical circuit alterations.
:::

## Issue Handling Procedure

<Steps>

1. Check the [Known Issues](/antbot/en/maintenance/known-issues/) page to see if the same issue has already been reported.

2. Review the [Error Codes](/antbot/en/maintenance/error-codes/) page for the cause and recommended actions for your error code.

3. If the issue remains unresolved, please submit it through [GitHub Issues](https://github.com/ROBOTIS-move/antbot/issues/new).

</Steps>
