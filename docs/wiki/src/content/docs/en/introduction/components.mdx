One AntBot unit and one manual charger are included by default. The wireless charging docking station is available as a separate purchase.

## AntBot Unit

4-Wheel Independent Swerve Drive robot platform with all sensors and onboard computer installed.

![AntBot Main Unit](../../../../assets/images/antbot_overview.png)

## Manual Charger

Connects to the [HMI panel](/antbot/en/hardware/hmi-panel/) charging port. See [Battery Charging](/antbot/en/quick-start/battery-charging/) for instructions.

![Manual Charger](../../../../assets/images/antbot_charger.png)

## Wireless Charging Docking Station (Optional)

Sold separately. Supports automatic docking and wireless charging. See [Battery Charging](/antbot/en/quick-start/battery-charging/#wireless-charging-station) for details.

![Charging Station](../../../../assets/images/antbot_charging_station.png)
