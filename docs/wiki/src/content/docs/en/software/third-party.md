These are external open-source and hardware manufacturer-provided packages used for operating AntBot.

| **Package**  | **Source** | **Description**                                        |
| ---------------- | ----------------- | ----------------------------------------------- |
| [DynamixelSDK](https://github.com/ROBOTIS-GIT/DynamixelSDK/tree/humble)     | ROBOTIS           | Standard SDK for Dynamixel Protocol 2.0 communication          |
| [coin_d4_driver](https://github.com/ROBOTIS-GIT/coin_d4_driver/tree/humble)   | ROBOTIS           | COIN-D4 2D LiDAR data reception and processing driver   |
| [vanjee_lidar_sdk](https://github.com/ROBOTIS-move/antbot/tree/main/vanjee_lidar_sdk) | Vanjee     | Vanjee 3D LiDAR ROS 2 standard driver             |
| [vanjee_lidar_msg](https://github.com/ROBOTIS-move/antbot/tree/main/vanjee_lidar_msg) | Vanjee     | Custom message definition package for Vanjee LiDAR data |
| [ublox](https://github.com/KumarRobotics/ublox/tree/ros2)            | KumarRobotics     | u-blox GPS/GNSS sensor ROS 2 driver             |
| [OrbbecSDK_ROS2](https://github.com/orbbec/OrbbecSDK_ROS2/tree/v2-main)   | Orbbec            | Orbbec RGB-D camera ROS 2 driver              |
