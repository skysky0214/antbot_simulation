This section will cover the simulation environment for testing AntBot software without physical hardware.

:::note
This guide is currently being written and will be provided in a future update. Track progress on the [GitHub repository](https://github.com/ROBOTIS-move/antbot).
:::

### Planned Content

- Gazebo-based physics simulation
- URDF/Xacro model visualization
- Sensor simulation (LiDAR, Camera, IMU)
- Nav2 integration testing

To view only the URDF model right now, run:

```bash
ros2 launch antbot_description description.launch.py
```

For installation and build instructions, see [Software Environment Setup](/antbot/en/software/environment-setup/).
