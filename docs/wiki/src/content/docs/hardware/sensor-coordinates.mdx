AntBot에 탑재된 주요 센서 목록과 `base_link` 기준 좌표입니다.

## 주요 센서 구성

| 센서 | 수량 | 연결 | 비고 |
| :--- | :---: | :--- | :--- |
| 3D LiDAR | 1 | Ethernet | |
| 2D LiDAR | 2 | USB | 전방 + 후방 장애물 감지 |
| RGB-D 카메라 | 1 | USB | Color + Depth |
| 모노 카메라 | 4 | V4L2 | 전방 3 + 후방 1 |
| IMU | 1 | USB | |
| GNSS | 1 | USB | GPS 수신기 |

---
