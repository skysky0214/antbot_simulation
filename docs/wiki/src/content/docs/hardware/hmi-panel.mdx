AntBot 좌측 하단에 HMI 패널이 있습니다.

![HMI 패널](../../../assets/images/antbot_hmi_panel.png)

---

## 포트 구성

| 번호 | 포트 | 용도 |
| :---: | :--- | :--- |
| 1 | HDMI | 외부 모니터 연결 |
| 2 | USB 1 (파란색) | 키보드 / 마우스 등 |
| 3 | USB 2 (흰색) | 내부 디버깅용 (사용 금지) |
| 4 | 수동 충전 단자 | 수동 충전기 연결 |

:::tip
일반 운행 시에는 HMI 패널을 사용할 필요가 없습니다. 디버깅이나 초기 설정 시에 활용합니다.
:::
