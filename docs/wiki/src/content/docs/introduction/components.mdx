AntBot 본체 1대와 수동 충전기 1개가 기본 제공됩니다. 무선 충전 도킹 스테이션은 별도 구매 옵션입니다.

## AntBot 본체

4륜 독립 조향 스워브 드라이브 로봇 플랫폼. 전체 센서와 온보드 컴퓨터가 탑재되어 있습니다.

![AntBot 본체](../../../assets/images/antbot_overview.png)

## 수동 충전기

[HMI 패널](/antbot/hardware/hmi-panel/) 충전 단자에 연결하여 충전합니다. 충전 방법은 [배터리 충전](/antbot/quick-start/battery-charging/)을 참조하세요.

![수동 충전기](../../../assets/images/antbot_charger.png)

## 무선 충전 도킹 스테이션 (옵션)

별도 구매. 자동 도킹 및 무선 충전을 지원합니다. 자세한 내용은 [배터리 충전](/antbot/quick-start/battery-charging/#무선-충전-스테이션)을 참조하세요.

![충전 스테이션](../../../assets/images/antbot_charging_station.png)
