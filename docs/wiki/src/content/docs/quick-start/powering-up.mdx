## 전원 켜기

<Steps>

1. AntBot 좌측 하단에 있는 **전원 버튼**을 **부저음이 들릴 때까지** 누릅니다. (약 1~2초)

2. LED가 점등되면서 **부팅이 시작**됩니다.

</Steps>

:::tip[다음 단계]
부팅이 완료되면 [네트워크 설정](/antbot/quick-start/network/)을 통해 SSH 접속을 진행하세요.
:::

---

## 전원 끄기

<Steps>

1. AntBot 좌측 하단에 있는 **전원 버튼**을 **부저음이 들릴 때까지** 누릅니다. (약 1~2초)

2. LED가 소등되면서 **로봇 전원이 꺼지게** 됩니다.

</Steps>
