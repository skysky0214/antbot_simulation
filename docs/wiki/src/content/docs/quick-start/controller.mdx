PS5 DualSense 컨트롤러를 사용하여 AntBot을 수동 조작할 수 있습니다.

![Controller Layout](../../../assets/images/joystick_dualsense.png)

| 번호 | 명칭 | 동작 | 비고 |
| :---: | :--- | :--- | :--- |
| 1 | L2 | 좌측 제자리 회전 | 정지 상태에서만, 트리거 방식 |
| 2 | L1 | 전조등 활성화 / 비활성화 | 토글 방식 |
| 3 | 좌측 스틱 | Y축: 전진/후진, X축: 커브 주행 | 최대 1.0 m/s |
| 4 | R2 | 우측 제자리 회전 | 정지 상태에서만, 트리거 방식 |
| 5 | R1 | 와이퍼 활성화 / 비활성화 | 토글 방식 |
| 6 | Triangle | 속도 레벨 UP | 1~9 단계 |
| 7 | Circle | 카고 잠금 해제 | |
| 8 | Cross | 속도 레벨 DOWN | 1~9 단계 |
| 9 | Square | 카고 잠금 | |

## 연결 방법

<Steps>

1. DualSense 컨트롤러를 **[USB 1 (파란색)](/antbot/hardware/hmi-panel/#포트-구성)** 에 유선 연결합니다.

2. `antbot` 서비스가 실행 중이라면 자동으로 컨트롤러를 인식합니다. (약 2~3초 소요)

   :::tip
   `antbot` 서비스는 부팅 시 자동으로 실행됩니다. 수동 실행이 필요한 경우 [소프트웨어 환경 구축 - 실행](/antbot/software/environment-setup/#실행)을 참고하세요.
   :::

3. 컨트롤러 LED가 점등되면 사용 준비 완료입니다.

</Steps>

## 속도 레벨

속도 레벨은 1~9 단계로 조절할 수 있으며, 기본값은 **3** 입니다.

| 레벨 | 최대 선속도 | 최대 각속도 |
| :---: | :---: | :---: |
| 1 | 0.11 m/s | 0.11 rad/s |
| 3 (기본) | 0.33 m/s | 0.33 rad/s |
| 5 | 0.56 m/s | 0.56 rad/s |
| 9 (최대) | 1.0 m/s | 1.0 rad/s |

## 키보드 원격 조종

컨트롤러가 없는 경우, SSH 접속 후 키보드로도 조종할 수 있습니다.

```bash
ros2 run antbot_teleop teleop_keyboard
```

| 키 | 동작 |
| :---: | :--- |
| `W` / `X` | 전진 / 후진 |
| `A` / `D` | 좌측 / 우측 이동 |
| `Q` / `E` | 반시계 / 시계 회전 |
| `1` ~ `9` | 속도 레벨 |
| `ESC` | 종료 |
